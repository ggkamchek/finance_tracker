# finance_tracker

Консольный трекер личных финансов на Go (офлайн, один пользователь, хранение в JSON).

## Возможности

- Добавление транзакций типов `доход` и `расход`
- Просмотр всех транзакций
- Редактирование и удаление по ID
- Фильтрация по категории
- Автоматический расчёт баланса
- Загрузка данных при старте и сохранение в `data/transactions.json`

## Допустимые категории

`еда`, `транспорт`, `зарплата`, `развлечения`, `здоровье`, `жилье`, `другое`

## Что нужно установить

- `Go` версии `1.22+` (рекомендуется актуальная стабильная версия)
- `Git` (по желанию, если нужен контроль версий)

Скачать Go можно с официального сайта: [https://go.dev/dl/](https://go.dev/dl/)

После установки проверьте в терминале:

```powershell
go version
```

Если команда `go` не находится, перезапустите терминал/IDE. На Windows также можно запускать через полный путь:

```powershell
& "C:/Program Files/Go/bin/go.exe" version
```

## Запуск

```powershell
go run .
```

Если `go` не добавлен в `PATH`:

```powershell
& "C:/Program Files/Go/bin/go.exe" run .
```

В консоли доступны команды: `помощь`, `добавить`, `список`, `изменить`, `удалить`, `фильтр`, `баланс`, `сохранить`, `выход`.
Для совместимости также работают английские команды: `help`, `add`, `list`, `edit`, `delete`, `filter`, `balance`, `save`, `exit`.
Дата вводится в формате `DD.MM.YYYY`.

## Тесты

```powershell
go test ./...
```

## Структура

- `main.go` - CLI и обработка команд
- `internal/transactions` - бизнес-логика (CRUD, фильтр, баланс, валидация)
- `internal/storage` - JSON persistence
- `docs/practice` - материалы для отчёта по практике
